HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element contains information about images that are displayed, including the alt attribute which provides text for accessibility and SEO.
  • The br element creates a line of empty space, or a line break.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add the working files to the staging area of Git; the -A flag indicates add all changes
  • git commit -m "message/description": commit the staged changes to our repository; -m flag associates a message with our commit
  • git pull origin main: syncs the local branch with the remote branch in GitHub; good habit to always pull the base branch into your feature branch before pushing up any code
  • git push orgin feature/name: push up all changes
